# 🔄 キャッシュクリア方法

## ✅ 拡張版は正常に動作しています！

サーバー側は正しく拡張版を提供していますが、ブラウザキャッシュのため古いページが表示されている可能性があります。

---

## 🌐 ブラウザキャッシュのクリア方法

### Google Chrome / Microsoft Edge

#### 方法1: ハードリロード（推奨）
1. **https://minecraft.schale41.jp** にアクセス
2. **Ctrl + Shift + R** (Windows) または **Cmd + Shift + R** (Mac) を押す

#### 方法2: 開発者ツールでクリア
1. **F12** キーを押して開発者ツールを開く
2. **Network** タブを選択
3. **Disable cache** にチェック
4. ページを **Ctrl + R** でリロード

#### 方法3: 完全なキャッシュクリア
1. **Ctrl + Shift + Delete** (Windows) または **Cmd + Shift + Delete** (Mac)
2. 「キャッシュされた画像とファイル」にチェック
3. 「データを削除」をクリック
4. ページをリロード

---

### Firefox

#### 方法1: ハードリロード
1. **https://minecraft.schale41.jp** にアクセス
2. **Ctrl + Shift + R** (Windows) または **Cmd + Shift + R** (Mac)

#### 方法2: キャッシュクリア
1. **Ctrl + Shift + Delete** (Windows) または **Cmd + Shift + Delete** (Mac)
2. 「キャッシュ」にチェック
3. 「今すぐ消去」をクリック
4. ページをリロード

---

### Safari (Mac)

#### 方法1: ハードリロード
1. **https://minecraft.schale41.jp** にアクセス
2. **Cmd + Option + R** を押す

#### 方法2: キャッシュクリア
1. **Safari** → **環境設定** → **詳細**
2. 「メニューバーに開発メニューを表示」にチェック
3. **開発** → **キャッシュを空にする**
4. ページをリロード

---

## 🔧 シークレットモード/プライベートブラウジング

キャッシュを気にせずテストする最も簡単な方法：

## 🔍 トラブルシューティング

### まだ古いページが表示される場合

#### 1. URLを直接指定
```
https://minecraft.schale41.jp/index-enhanced.html
```
拡張版HTMLファイルに直接アクセスして確認

#### 2. クエリパラメータを追加
```
https://minecraft.schale41.jp?v=3.0
```
URLにバージョンパラメータを追加してキャッシュ回避

#### 3. 別のブラウザでテスト
- 普段使わないブラウザでアクセス
- モバイルデバイスでアクセス

#### 4. DNSキャッシュもクリア (オプション)

**Windows:**
```cmd
ipconfig /flushdns
```

**Mac / Linux:**
```bash
sudo dscacheutil -flushcache
sudo killall -HUP mDNSResponder
```

## 📱 モバイルでのキャッシュクリア

### iOS (Safari)
1. **設定** → **Safari**
2. **履歴と Web サイトデータを消去**
3. Safari を再起動
4. https://minecraft.schale41.jp にアクセス

### Android (Chrome)
1. **Chrome** → **︙** (メニュー)
2. **設定** → **プライバシーとセキュリティ**
3. **閲覧履歴データの削除**
4. 「キャッシュされた画像とファイル」を選択
5. **データを削除**
6. https://minecraft.schale41.jp にアクセス
