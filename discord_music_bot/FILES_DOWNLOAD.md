## 🚀 使用開始手順

### 1. ファイルのダウンロード
上記のZIPファイルをダウンロードして展開

### 2. 依存関係のインストール
```bash
pip install -r requirements.txt
```

### 3. 環境変数の設定
```bash
cp .env.example .env
# .envファイルを編集してDISCORD_TOKENを設定
```

### 4. 実行
```bash
python run.py
```
